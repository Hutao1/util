想着做一个工具合集，让自己工作中或者生活中能够方便一些，工具只做纯前端，依赖于github上使用，不然放在自己服务器上，一两年又要换一次服务器，太过麻烦

因为用的免费接口，所以返回时间可能较长，请耐心等待下

[工具合集地址](https://hutao1.github.io/util/dist/#/)

```js

```

输出:

```js

 freeInterface: null, // 免费接口

}
```
